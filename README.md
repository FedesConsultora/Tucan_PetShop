# Tucan - Pet Shop de La Plata

Este proyecto es la página de aterrizaje oficial de Tucan, una tienda de mascotas dedicada a servir a la comunidad de La Plata.

## Desarrollo

Desarrollado con el apoyo de **SedeF**.
Para más información, visita [fedes.ai](https://fedes.ai).

## Stack Tecnológico

- **Framework**: Vite
- **Estilos**: SCSS (utilizando `@use` y `sass:color` moderno)
- **Desarrollador**: SedeF
